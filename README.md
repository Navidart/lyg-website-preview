# Luxury Yacht Group Landing Preview

React/Vite frontend prototype for the Luxury Yacht Group landing page mockup.

## Install

```bash
npm install
```

## Run Locally

```bash
npm run dev
```

Vite will print a local URL, usually `http://localhost:5173`.

## Build

```bash
npm run build
```

The production files are generated in `dist/`.

## Preview Build

```bash
npm run preview
```

## Share With a Client

Run `npm run dev` or `npm run preview`, then share the local URL during a screen share. For a packaged handoff, run `npm run build` and host the generated `dist/` folder on any static hosting provider.

## Notes

- This is a frontend-only prototype. There is no backend.
- The layout is component-based and responsive for desktop, tablet, and mobile.
- Figma-provided image URLs are used where no local `project/assets` files were available in this workspace.
